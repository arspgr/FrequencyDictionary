Входные параметры: {полное имя файла для анализа} {полное имя файла вывода}.
Второй параметр необязательный: в случае, если он не передан, используется путь по умолчанию и название файла result.txt.

MainService - основной сервис, отправная точка запуска других сервисов. 
IFileProcessor - процессор обработки файла, использует для обработки строки ILineProcessor.
IFileWriterService - сервис записи результата в файл.
